# Competição Kaggle - Matemática

Workspace dedicado à competição de machine learning.

## Estrutura do Projeto

```
comp-kaggle-matematica/
├── data/              # Dados brutos e processados
├── notebooks/         # Jupyter notebooks para análise exploratória
├── src/               # Código-fonte (funções reutilizáveis)
├── models/            # Modelos treinados (.pkl, .pth, etc)
├── results/           # Resultados finais e submissions
├── requirements.txt   # Dependências do projeto
└── README.md          # Este arquivo
```

## Setup Inicial

1. Crie um ambiente virtual:
```bash
python -m venv venv
venv\Scripts\activate
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

3. Adicione dados ao diretório `data/`

## Development

- Use `notebooks/` para análise exploratória
- Modularize código reutilizável em `src/`
- Salve modelos treinados em `models/`
- Armazene resultados finais em `results/`

Good luck! 🚀
